- International broadcasters (VOA, BBC, etc.)
- Professional HF system designers
- Serious amateur radio operators

INSTALLATION - WINDOWS
=======================

Step 1: Install pythonprop
---------------------------
Open Command Prompt or PowerShell:

cd "D:\Python Scripts\Ham"
pip install pythonprop

Note: pythonprop is a Python wrapper for VOACAP. It includes the VOACAP
      executable and coefficient files.

Step 2: Verify Installation
----------------------------
python -c "import pythonprop; print('VOACAP ready!')"

If you see "VOACAP ready!" then you're all set!

Step 3: Test VOACAP Mode
-------------------------
python generate_propagation_voacap.py --voacap

This will generate predictions using VOACAP instead of ITU-R.
First run may take 30-60 seconds.

TROUBLESHOOTING
===============

Error: "pythonprop not found"
-----------------------------
Solution: pip install pythonprop

Error: "VOACAP executable not found"
------------------------------------
pythonprop should include the executable. Try:
  pip uninstall pythonprop
  pip install pythonprop --no-cache-dir

Error: "Coefficient files missing"
----------------------------------
pythonprop includes these. If missing:
  1. Download from: https://www.voacap.com/
  2. Place in pythonprop installation directory

Error: "Predictions taking too long"
------------------------------------
VOACAP is computationally intensive. Normal timing:
  â€¢ Single prediction: 0.5-1 second
  â€¢ 70 predictions: 30-60 seconds
  â€¢ First run: May take longer (initialization)

If consistently >2 minutes, check:
  â€¢ CPU usage (should be high during generation)
  â€¢ Disk speed (SSD recommended)
  â€¢ Python version (3.8+ recommended)

TECHNICAL DETAILS
=================

VOACAP Calculation Method:
---------------------------
1. Ionospheric Coefficients
   - CCIR/URSI monthly coefficient files
   - Solar activity correlation
   - Geomagnetic activity effects

2. Ray Tracing
   - Multi-hop propagation paths
   - E/F layer reflection points
   - Ground reflection losses

3. Signal Strength
   - Transmit power
   - Antenna gain/pattern
   - Path loss calculation
   - Receiver noise floor

4. Reliability
   - Statistical methods
   - Upper/lower decile predictions
   - Time probability analysis

ITU-R Calculation Method:
--------------------------
1. Critical Frequency
   - Simple foF2 model
   - Solar flux correlation
   - Diurnal variation

2. MUF Calculation
   - Hop geometry estimate
   - Single calculation
   - No ray tracing

3. Signal Quality
   - FOT/MUF ratio
   - Simple absorption model
   - Statistical reliability

Accuracy Expectations:
----------------------
Simple paths (1-3000 km, mid-latitude):
  ITU-R:  75-85% accuracy
  VOACAP: 90-95% accuracy

Complex paths (>5000 km, trans-equatorial):
  ITU-R:  65-75% accuracy
  VOACAP: 85-92% accuracy

Auroral/disturbed conditions:
  ITU-R:  60-70% accuracy
  VOACAP: 80-88% accuracy
