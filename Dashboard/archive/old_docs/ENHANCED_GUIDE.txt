ADVANCED TIPS:
--------------

GRAY LINE HUNTING:
- Gray line is the transition zone between day/night
- Shown as yellow shaded area on map
- BEST times are when target is in gray line
- OR when YOU are in gray line
- Morning gray line: 30 min before to 30 min after sunrise
- Evening gray line: 30 min before to 30 min after sunset
- Long-path signals often peak at gray line

MUF OPTIMIZATION:
- Don't operate AT the MUF - band will be unstable
- Best frequency = 85% of MUF (shown as "Best: X MHz")
- If MUF is 14.5 MHz, operate around 12.3 MHz
- Higher in band = fewer hops = stronger signal (if below MUF)

S-METER INTERPRETATION:
- S7+ = Good for SSB
- S5+ = Good for CW/Digital
- S3-S4 = CW/Digital only, difficult
- FT8 can work down to S1 or less!

RELIABILITY PERCENTAGE:
- >70% = Excellent, high probability of QSO
- 50-70% = Good, should be workable
- 30-50% = Marginal, try CW or digital
- <30% = Poor, long shots only

MULTI-DAY PLANNING:
- Check 72-hour forecast before contests
- Plan operating schedule around peak times
- Watch for solar changes (Kp index rising)
- Gray line windows repeat daily Â±4 minutes

SOLAR EVENT RESPONSES:
- Kp >5 = Work lower bands (40m/30m better than 20m+)
- Low SFI = Focus on 40m/30m, higher bands may be dead
- After solar flare = Wait 1-2 days, check Kp/A index
- CME impact = Aurora possible, try 6m if aurora visible

TROUBLESHOOTING
===============

MAP NOT SHOWING:
- Make sure you're using: http://localhost:8000/
- Not: file:/// protocol
- Gray line requires internet for leaflet-terminator library

NO DXCC ALERTS:
- Make sure dxcc_summary.json is in same folder
- Alerts only show when needed entities are workable
- Check that you have needed entities in list

TIMELINE TOO LONG:
- 72 hours is a lot of data
- Scroll the timeline box
- First 24h shown every 2 hours
- Days 2-3 shown every 6 hours only

PAGE TOO WIDE:
- Designed for 1600px width
- Resize browser window to fit
- Below 1400px, layout switches to single column
- Use Ctrl+- to zoom out if needed

OUTDATED PREDICTIONS:
- Remember to regenerate!
- python generate_propagation.py
- Refresh browser (F5 or Ctrl+R)
- Check "Last Updated" time at bottom
