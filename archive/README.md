## Archive Policy

Files are moved to this archive when:
1. Investigation is complete and findings are incorporated into code
2. Weekly/phase milestone is finished
3. Content is superseded by newer documentation
4. Historical reference value but not actively consulted

Files are **not** archived if:
- Still actively referenced in development workflow
- Contain current validation results or status
- Part of contributor onboarding process
